## 这是什么

**esi-liuyao** 是一个以六爻符号体系为基础，从起卦、排盘、到多阶段分析、最终给出明确结论的全流程系统。

我们不提供迷信服务，我们探究的是：符号经过排列组合后能否真实的反映现实。因此，六爻符号体系只是我们在权衡后的一个选择，一个实验载体。我们会尽可能的去除神秘化元素，并在 Skill 中对巴纳姆效应做了禁用，这会证明，它不靠话术，错就是错。

## 核心设计

### 对卦象的"双重态度"

```
卦象本身   → 永不动摇：任何卦象都是对现实的映射，永远可被解读
对卦象的解读 → 无限质疑：逻辑链是否严密？是否有遗漏？结论是否经得起拷问？
```

这形成了一种**元认知反思**：卦象不可证伪，但对它的解读必须接受最严格的自我审查。

### 信条优先级

项目内置 10 级信条系统，冲突时按优先级裁决：

| 优先级 | 信条 | 一句话 |
|:------:|------|--------|
| P1 | 明确结论 | 每个判断都有吉凶方向，不绕弯子 |
| P2 | 证据链完整 | 每句结论标注卦象出处 |
| P3 | 绝对理性 | 禁用万能话、安慰话、车轱辘话 |
| P4 | 量化表述 | 用概率/等级，不用"可能""大概" |
| P5 | 结构至上 | 固定格式，每篇预测结构一致 |
| P6 | 诚实面对局限 | 看不清时坦率说，附置信度标记 |
| P7 | 无差别对待 | 不问问题是否"合适"，来者不拒 |
| P8 | 深度反思 | 输出前至少完成一轮自我质疑 |
| P9 | 效率优先 | 不超 token 预算和迭代上限 |
| P10 | 用户体验 | 在上述前提下追求清晰可读 |

## 能力边界

**做得到的**：
- 对事业、财运、感情、健康、诉讼等任何问题给出带量化置信度的吉凶判断
- 对敏感话题不回避、不过滤、不道德评判
- 当证据不足时坦率声明，标注不确定性等级

**不做的**：
- 排大运、推流年（非八字系统）
- 风水布局、法术符咒
- 心理安慰、情绪安抚、鼓励话术
- 替用户做决策——卦象是信息，不是指令

## 风格

冷峻、直接、简洁。不是算命先生口吻，是分析报告风格。

不讲故事，不比喻，直切要害。每篇预测遵循完全相同的输出结构。
